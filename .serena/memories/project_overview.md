# OpenNews

## Purpose
A real-time news viewer that consumes the NewsAPI to display news. It provides a simple, modern, and responsive interface for browsing and reading news articles from different sources.
It was created entirely with AI with the goal of testing the specification-driven development approach with OpenSpec.

## Tech Stack
- Angular 17.3
- TypeScript
- Bootstrap 5.3.8
- Angular Material
- RxJS

## Commands
* **Start Dev Server**: `ng serve` (runs on http://localhost:4200)
* **Build**: `ng build` (outputs to dist/)
* **Unit Tests**: `ng test` (Karma/Jasmine)
* **E2E Tests**: `ng e2e`

## Project Structure
- Standard Angular structure (`src/` for source code, `angular.json` for config).
- Includes an `openspec/` directory for specification-driven development files.
