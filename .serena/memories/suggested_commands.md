# Commands to run in this codebase

- Start development server: `ng serve` or `npm start`
- Build the project: `ng build`
- Run unit tests: `ng test`
- Generate components: `ng generate component <name>`
- Generate services: `ng generate service <name>`

As the system is Windows, use standard Windows paths and commands (or PowerShell equivalents). You should use `uv run` to interact with Python tools if necessary, but this is an Angular project so `npm` and Angular CLI (`ng`) are primary.
